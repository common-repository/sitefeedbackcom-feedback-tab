== Description ==

This plugin lets you add a feedback tab at the bottom of your site pages to collect user feedback. Ratings and comments are collected on http://www.sitefeedback.com where the site owner can review, manage, and interact with user comments.

You are not required to have an account with http://www.sitefeedback.com; however, an account will let you access advanced features that let you hide and promote comments, as well as turn the comment into a to do list.

To see example of sites that are currently using our plugin, please visit http://www.sitefeedback.com/topsites/

== Installation ==

1. Upload `sitefeedbackcom-feedback-tab` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

That's it. You should see our feedback tab showing up at the bottom of your pages.
